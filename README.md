# Bộ xây dựng giao diện HTML CSS siêu tốc
Công nghệ sử dụng `Gulp` + `Pug` + `SCSS` + `BrowserSync`

## Cài đặt
Yêu cầu: NodeJS, GULP
NodeJS: `https://nodejs.org/`
GULP: `npm install --global gulp-cli`
Chạy lệnh `npm install` hoặc `yarn install`

## Bắt đầu
Chạy lệnh: `gulp`
